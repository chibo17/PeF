Dragi študentje/študentke, dobrodošli pri predmetu APS. Na tem github repozitoriju boste našli navodila za vaje in druge materiale potrebne za študij. 

## Struktura repozitorija

Ta repozitorij je organiziran na naslednji način:

- **vaje/**: V tej mapi boste našli vse vaje, razdeljene po tednih.
- **predavanja/**: Tukaj so shranjene predstavitve in dodatni materiali, ki so bili uporabljeni med predavanji.
- **gradiva/**: Dodatna gradiva, kot so članki, knjige in povezave do uporabnih virov, so shranjena tukaj.
- **README.md**: Ta datoteka vsebuje osnovne informacije o repozitoriju in navodila za uporabo.

Upamo, da vam bo ta repozitorij v pomoč pri učenju in razumevanju algoritmov in podatkovnih struktur. Če imate kakršna koli vprašanja ali potrebujete pomoč, se obrnite na  profesorja.
